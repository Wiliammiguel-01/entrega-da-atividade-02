# Sem Dono? Sem Problemas

## Descrição
Este projeto é uma aplicação web SPA (Single Page Application) que demonstra:  
- Navegação entre páginas sem recarregar (SPA navigation)  
- Layout responsivo para diferentes tamanhos de tela  
- Estrutura organizada com HTML, CSS e JS  
- Funcionalidades básicas de formulário e rodapé  

## Tecnologias Utilizadas
- HTML5  
- CSS3 (com variáveis e layout modular)  
- JavaScript (ES6+)  

## Estrutura do Projeto
/css       → Arquivos de estilo
/js        → Scripts da aplicação
/images    → Imagens utilizadas
index.html → Página inicial

## Funcionalidades
- Navegação SPA entre páginas (Home, Projetos, Contato)  
- Layout responsivo  
- Formulário de contato com validação básica  
- Rodapé com informações de contato e endereço
